# Мой любимый проект — Weather Bot 🌤️

## Описание

**Weather Bot** — это телеграм-бот, который показывает текущую погоду в любом городе мира.  
Написан на _Python_ с использованием библиотеки `python-telegram-bot` и API сервиса OpenWeatherMap.

## Что умеет бот

- Показывать текущую температуру и ощущаемую температуру
- Выводить описание погодных условий
- Отправлять прогноз на 5 дней вперёд
- Поддерживать несколько языков (русский и английский)

## Технологии

| Технология          | Назначение           |
| ------------------- | -------------------- |
| Python 3.11         | Основной язык        |
| python-telegram-bot | Telegram API обёртка |
| OpenWeatherMap API  | Данные о погоде      |
| dotenv              | Управление секретами |

Это был мой **первый реальный проект**, который я сделал не по туториалу.
